This project is a Google Translator desktop application built using Python GUI.
It allows users to translate text from one language to another quickly and easily. 
The application provides a simple and user-friendly interface where users can enter text, select the target language, and instantly get the translated result.
The project uses Tkinter for the graphical user interface and Google Translator API (deep_translator) for performing language translations.

✨ Features

🌐 Translate text into multiple languages
🖥️ Simple and user-friendly GUI
🔤 Automatic language detection
⚡ Fast translation using Google Translate API
📋 Separate text areas for input and translated output
🎯 Language selection using dropdown (ComboBox)
